Smoke test from inside the cluster:

  kubectl -n {{ .Release.Namespace }} run kvbctl --rm -it --restart=Never \
    --image={{ include "kvblockd.image" . }} --command -- /kvblockd --version

  # with a kvbctl binary:
  kvbctl put -addr {{ include "kvblockd.fullname" . }}:{{ .Values.service.dataPort }} \
    -ns demo -token demo-token hello - <<< "world"

Remember:
- The demo namespace/token in values.yaml is a quickstart credential —
  change or remove it before pointing real traffic here.
- The data port speaks the KVB1 wire protocol for same-AZ/VPC clients;
  never expose it to the internet.
- Grafana: import deploy/grafana/kvblockd-dashboard.json and scrape
  port {{ .Values.service.metricsPort }}.
