# Security Policy

kvblockd is pre-v0.1 and NOT production software yet.

To report a vulnerability, email rudraymehra@gmail.com with subject
"[kvblockd security]". Please do not open public issues for security
reports. We aim to acknowledge within 72 hours.

Design notes relevant to security (multi-tenant isolation, hash-flood
resistance, cross-tenant side channels) live in docs/DESIGN.md as they land.
